# SimpleTimer - シンプルなPHPタイマーアプリ

このタイマーアプリケーションは、ブラウザで動作するシンプルなカウントダウンタイマーです。

------------------------------------------------------------------------

## Windows での手順

Windowsでは、以下の2つの方法があります。

------------------------------------------------------------------------

# 方法1：HTA版を直接実行する（簡単）

1.  `SimpleTimer.hta` をダブルクリックします。
2.  タイマーが起動します。

------------------------------------------------------------------------

# 方法2：XAMPPを使ってサーバー上で動かす

## 1. XAMPP をダウンロード・インストールする

1.  XAMPP公式サイトにアクセスします\
    https://www.apachefriends.org/jp/index.html
2.  Windows用のXAMPPをダウンロードします。
3.  ダウンロードしたファイルを実行します。
4.  インストール時の質問は基本的に「次へ」で進みます。

------------------------------------------------------------------------

## 2. XAMPPを起動する

1.  「XAMPPコントロールパネル」を開きます。
2.  「Apache」の横にある「Start」ボタンをクリックします。
3.  Webサーバーが起動します。

------------------------------------------------------------------------

## 3. タイマーファイルを配置する

1.  `timer.php` ファイルをコピーします。
2.  エクスプローラーで次のフォルダを開きます。

C:`\xampp`{=tex}`\htdocs`{=tex}\

3.  そのフォルダ内に `timer.php` を貼り付けます。

------------------------------------------------------------------------

## 4. タイマーを使う

1.  ブラウザ（Chrome / Edge など）を開きます。
2.  アドレスバーに次のURLを入力します。

http://localhost/timer.php

3.  タイマーが表示され、使用できるようになります。

------------------------------------------------------------------------

以上でセットアップ完了です。
